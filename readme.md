# Juego del Ahorcado

Este es un sencillo juego del ahorcado desarrollado en PHP. Puedes jugar adivinando letras hasta completar la palabra secreta. Dispones de 6 intentos antes de perder.

## Cómo jugar
1. Introduce una letra en el formulario.
2. Comprueba si la letra está en la palabra secreta.
3. Sigue jugando hasta adivinar toda la palabra o perder todas tus vidas.

## Instrucciones para desarrolladores
- Los estilos están definidos en `estilos.css`.
- Las palabras disponibles están definidas en el archivo `index.php`.
